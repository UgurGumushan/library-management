# Library Management

## Prerequisites
- Node.js 25+
- pnpm

## Setup
1. Clone the repository.
2. Install dependencies:
   
   ```bash
   pnpm install
   ```
3. Create a `.env` file with:
   
   ```
   DATABASE_URL=<Provided Separately>
   PORT=3000
   TEST_PORT=3001
   ```

## Run
```bash
pnpm start
```

## Test
```bash
pnpm test
```
